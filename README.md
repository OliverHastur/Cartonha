# Uma Carta Através do Tempo

> "Amei-te desde o primeiro instante e amar-te-ei até o último. Talvez até depois." — **Ian Clarke**.

Este projeto é uma experiência digital imersiva criada como um presente especial. Unindo o romantismo clássico do livro Perdida de Carina Rissi com o cenário icônico de minha terra: Salvador, Bahia.

## 🌟 O Conceito
Uma releitura da carta de Ian Clarke adaptada para o meu presente, celebrando uma história que começou em 2017 e se mantém viva através da amizade e das memórias. Produzido sob o selo fictício **[NOME DA PRODUTORA DELA]**, em conexão com a marca **Coqueteria**.

## 🛠️ Funcionalidades Técnicas.
* **Efeito Typewriter:** Texto datilografado em "tempo real" com pausas humanas para efeito de realidade.
* **Auto-Scroll:** Sistema que acompanha a leitura automaticamente, garantindo uma experiência cinematográfica.
* **Interatividade:** * Cursor personalizado.
    * Glossário de memórias via *hover* em palavras-chave (Funcionalidade exclusiva para o computador).
    * Mia, a gata interativa, com balão de fala e áudio.
* **Gestão de Consentimento:** Uma tela inicial que respeita o espaço e a vontade da leitora antes de iniciar a experiência.

## 🧰 Tecnologias Utilizadas
* **HTML5 / CSS3**.
* **JavaScript Vanilla**.
* **GSAP (GreenSock Animation Platform)**.
* **Google Fonts:** Utilização das fontes *Inter* e *Pinyon Script* para o contraste entre o moderno e o clássico.
